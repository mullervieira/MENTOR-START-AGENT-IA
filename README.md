# Mentor Start

O Mentor Start é um agente de IA para quem está começando em tecnologia. Ele explica conceitos, ajuda a organizar os primeiros estudos e oferece dicas graduais para exercícios, sem entregar respostas prontas de atividades avaliativas.

## Estrutura

- `AGENTS.md`: regras principais do agente.
- `agent/`: persona e base de conhecimento.
- `skills/`: processos para tarefas recorrentes.
- `docs/`: documentação da entrega.
- `src/`: futura aplicação funcional.

## Objetivo da primeira versão

O agente deve explicar conceitos simples, sugerir um próximo passo e reconhecer quando sua base não for suficiente. A base é pequena de propósito: ela será ampliada após pesquisa e validação.
