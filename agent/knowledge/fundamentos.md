# Fundamentos de programação

Esta é uma base inicial. Amplie cada tópico somente depois de pesquisar e revisar o conteúdo.

## Algoritmo

Um algoritmo é uma sequência de passos para resolver um problema. Uma receita é uma analogia: ingredientes são entradas, preparo são passos e o prato pronto é o resultado.

## Variável

Uma variável é um nome usado para guardar um valor que pode ser consultado ou alterado durante um programa.

## Condição

Uma condição permite que o programa tome caminhos diferentes conforme uma pergunta seja verdadeira ou falsa.

## Repetição

Uma repetição executa uma tarefa mais de uma vez, útil para aplicar a mesma ação a vários itens.

## Função

Uma função agrupa uma tarefa reutilizável, ajudando a organizar e evitar repetição de lógica.
