# Ferramentas iniciais

## Editor de código

Um editor de código é o programa usado para escrever e organizar arquivos de um projeto. O VS Code é um exemplo popular.

## Terminal

O terminal permite interagir com o computador por comandos, como navegar por pastas e executar programas.

## Git

Git registra o histórico de alterações de um projeto. Ele permite entender o que mudou e recuperar versões anteriores quando necessário.

## GitHub

GitHub é uma plataforma online que hospeda repositórios Git e ajuda a compartilhar código, colaborar e montar portfólio.
