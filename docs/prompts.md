# Prompts do Mentor Start

`AGENTS.md` é a fonte principal das instruções do agente.

## Estrutura de resposta

1. Regras e persona.
2. Trechos relevantes da base.
3. Pergunta do estudante.
4. Resposta simples, honesta e adequada ao nível iniciante.

## Casos para testar

- "O que é Git?"
- "Quero começar a estudar programação; por onde vou?"
- "Travei em um exercício de condições."
- Uma pergunta fora da base, para verificar se o agente reconhece a limitação.
