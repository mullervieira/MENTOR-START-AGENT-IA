# Pitch final — Mentor Start

## Roteiro de até 3 minutos

### Problema — 30 segundos

"Quem começa em tecnologia encontra muitas linguagens, ferramentas e trilhas ao mesmo tempo. Isso gera dúvidas básicas, insegurança e a sensação de não saber por onde começar. Além disso, uma IA que só entrega respostas prontas pode atrapalhar o aprendizado."

### Solução — 35 segundos

"Por isso criei o Mentor Start, um agente de IA educacional para iniciantes em tecnologia. Ele explica conceitos com linguagem simples, ajuda a criar planos de estudo possíveis e orienta exercícios com dicas graduais, sem fazer a atividade no lugar do estudante."

### Como funciona — 45 segundos

"O projeto tem quatro camadas. O AGENTS.md define regras, limites e tom. A persona define a postura pedagógica. A base de conhecimento reúne fundamentos, ferramentas e trilhas. As skills organizam como o agente explica conceitos, monta planos e ajuda em exercícios."

### Demonstração — 45 segundos

"Na aplicação, a pessoa pergunta: 'Qual a diferença entre Git e GitHub?'. O Mentor Start consulta a base local, explica os dois conceitos, mostra um exemplo e sugere uma prática. Se a pessoa pedir uma solução pronta para uma atividade, o agente recusa com respeito e oferece orientação gradual."

### Avaliação e diferencial — 35 segundos

"Eu avaliei o agente com casos de explicação, planejamento, exercícios, segurança digital e perguntas fora da base. Os critérios são clareza, fidelidade ao conteúdo, segurança, autonomia e utilidade. O diferencial do Mentor Start não é responder tudo: é reconhecer limites e transformar dúvidas em próximos passos possíveis."

### Encerramento — 20 segundos

"Além de ensinar tecnologia, o Mentor Start também reforça hábitos de cibersegurança, como proteger chaves de API e reconhecer phishing. Como evolução, pretendo ampliar a base, melhorar a recuperação de contexto e adicionar pesquisa web com fontes verificáveis."

## Demonstração recomendada

Use a pergunta: **"Nunca programei. Qual a diferença entre Git e GitHub?"**.
