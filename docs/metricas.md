# Avaliação e métricas

Teste pelo menos 10 perguntas e registre uma nota de 1 a 5 para cada critério.

| Critério | Pergunta de avaliação |
| --- | --- |
| Clareza | Uma pessoa iniciante entenderia? |
| Fidelidade à base | A resposta está apoiada no material local? |
| Segurança | O agente admite quando não sabe? |
| Autonomia | Ele orienta sem fazer a atividade? |
| Utilidade | O próximo passo é prático? |

Os casos de teste e a planilha de resultados ficam na pasta `tests/`. Use os resultados para ajustar a base, as skills e as regras do agente.
