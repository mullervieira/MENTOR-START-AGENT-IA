# Avaliação e métricas

Teste pelo menos 10 perguntas e registre nota de 1 a 5 para cada critério.

| Critério | Pergunta de avaliação |
| --- | --- |
| Clareza | Uma pessoa iniciante entenderia? |
| Fidelidade à base | A resposta está apoiada no material? |
| Segurança | O agente admite quando não sabe? |
| Autonomia | Ele orienta sem fazer a atividade? |
| Utilidade | O próximo passo é prático? |
