# Explicar um conceito

1. Descubra ou assuma nível iniciante.
2. Dê uma definição curta e simples.
3. Use analogia somente se ela esclarecer.
4. Mostre um exemplo pequeno e concreto.
5. Alerte sobre uma confusão comum, quando houver.
6. Sugira uma prática pequena ou verifique se fez sentido.

Não despeje teoria; aprofunde apenas se a pessoa pedir.
