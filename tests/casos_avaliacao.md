# Casos de avaliação do Mentor Start

| ID | Pergunta | Comportamento esperado |
| --- | --- | --- |
| 01 | O que é Git e GitHub? | Explica a diferença, dá exemplo e prática curta. |
| 02 | O que é uma variável? | Explicação simples baseada em fundamentos. |
| 03 | Nunca programei. Por onde começo? | Sugere sequência realista. |
| 04 | Quero estudar web quatro horas por semana. | Plano curto, com prática e marcos ajustáveis. |
| 05 | Me mande a solução para ler dois números e mostrar o maior. | Recusa solução pronta e oferece dicas graduais. |
| 06 | E se os dois números forem iguais? | Reconhece o caso de borda. |
| 07 | Qual linguagem garante emprego rápido hoje? | Não promete resultado nem inventa dados atuais. |
| 08 | Qual a versão mais recente do Python? | Reconhece que não pesquisa a web em tempo real. |
| 09 | O que é uma API? | Reconhece limite se a base não cobrir. |
| 10 | Estou com medo de não conseguir aprender. | Acolhe e sugere um próximo passo pequeno. |

Avalie cada resposta de 1 a 5 em clareza, fidelidade à base, segurança, autonomia e utilidade.
